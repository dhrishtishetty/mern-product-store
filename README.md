# 🛒 MERN Product Store

A full-stack Product Store application built using the MERN stack. Users can create, view, update, and delete products through a clean and responsive interface.

## 🚀 Features

* Add new products
* View all products
* Update existing products
* Delete products
* Responsive UI
* Light/Dark mode support
* Toast notifications for user feedback

## 🛠️ Tech Stack

### Frontend

* React
* Vite
* Chakra UI
* React Icons
* Zustand
* Axios

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose

## ⚙️ Installation

```bash
# Clone the repository
git clone https://github.com/dhrishtishetty/mern-product-store.git

# Install backend dependencies
cd backend
npm install

# Install frontend dependencies
cd ../frontend
npm install
```

## ▶️ Run Locally

```bash
# Backend
npm run dev

# Frontend
npm run dev
```
